# Video Game Sales Analysis

## 📌 Project Description
The goal of this project is to analyze global video game sales data to identify trends and factors that contribute to a game's commercial success.

The analysis focuses on understanding how platform, genre, critic scores, and regional preferences influence sales performance. These insights can help inform marketing strategies and future product development decisions.

---

## 📊 Data Description
The dataset includes information on video games such as:

- Game title and release year
- Platform and genre
- Sales across different regions (North America, Europe, Japan, etc.)
- Critic and user scores

---

## ⚙️ Methods and Approach

### Data Preprocessing
- Handled missing values and inconsistent data
- Converted data types where necessary
- Filtered relevant time periods for analysis

### Exploratory Data Analysis (EDA)
- Analyzed overall sales trends over time
- Examined platform popularity and lifecycle
- Investigated genre performance

### Statistical Analysis
- Explored relationships between critic scores and sales
- Compared regional differences in game preferences
- Identified key drivers of success

### Visualization
- Created charts to highlight trends and patterns
- Used visual analysis to support findings

---

## 📈 Results

- Certain platforms and genres consistently outperform others
- Critic scores show a relationship with sales, especially for high-performing games
- Regional differences play a significant role in game success

---

## 💡 Conclusion

This project demonstrates how data analysis can provide valuable insights into market trends and consumer behavior.

These insights can help:
- Guide marketing and release strategies
- Inform game development decisions
- Target specific regional markets more effectively
